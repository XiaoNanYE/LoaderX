# SuperBigLoaderX - a BDS Plugin Loader

> SuperBigLoaderX是一个基于 Bedrock Server 的重量级插件加载器

##### 1.我为什么要使用它？
**SuperBigLoaderX** 可以使用 Java，JavaScript，Python，C，C++，Lua，.NET，易语言。等编程语言进行插件编写。
不仅如此，SuperBigLoaderX还支持自动注入BDS，省去了小麻烦！
